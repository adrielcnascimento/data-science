# 📊 Data Science

Repositório com exercícios, experimentos e projetos desenvolvidos ao longo da minha jornada com **Ciência de Dados**, **Mineração de Dados** e análise de **dados públicos**.

---

## 🗂️ Estrutura do Repositório

```
data-science/
├── datasets/          # Conjuntos de dados utilizados nos projetos e análises
└── machine-learning/  # Notebooks de experimentos e modelos de Machine Learning
```

---

## 🛠️ Tecnologias e Ferramentas

| Ferramenta / Biblioteca | Uso |
|---|---|
| **Python** | Linguagem principal dos notebooks |
| **Jupyter Notebook** | Ambiente de desenvolvimento e experimentação |
| **Pandas** | Manipulação e análise de dados |
| **NumPy** | Computação numérica |
| **Scikit-learn** | Algoritmos de Machine Learning |
| **Matplotlib / Seaborn** | Visualização de dados |

---

## 📁 Descrição das Pastas

### `datasets/`
Contém os conjuntos de dados (datasets) utilizados nos projetos. Inclui dados públicos coletados e organizados para uso nas análises e modelos.

### `machine-learning/`
Notebooks Jupyter com experimentos, implementações e estudos de algoritmos de Machine Learning aplicados à mineração e análise de dados.

---

## 🎯 Conteúdo Abordado

- Análise exploratória de dados (EDA)
- Pré-processamento e limpeza de dados
- Mineração de dados (*Data Mining*)
- Algoritmos de Machine Learning (classificação, regressão, clusterização)
- Visualização e interpretação de resultados
- Trabalho com dados públicos

---

## 🎓 Sobre

Este repositório reúne exercícios e experiências práticas com Ciência de Dados, com foco em Mineração de Dados e exploração de conjuntos de dados públicos.
